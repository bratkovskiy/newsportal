# SEO Templates - Примеры работы

## Примеры генерации meta-тегов

### 1. Главная страница (150 статей)

**Входные данные:**
- Количество статей: 150

**Результат:**
```html
<title>Последние новости сегодня — 150 материалов | FashionSite</title>
<meta name="description" content="Актуальные события, аналитика и топ-материалы: 150 свежих новостей в России и мире на FashionSite.">
```

---

### 7. Статья - Длинный excerpt (более 160 символов)

**Входные данные:**
- Заголовок: "Как изменилась индустрия моды"

**Результат:**

## Логика обработки excerpt для статей

### Алгоритм:

1. **Пустой excerpt** → `"Подробности — на {Бренд}."`

2. **Есть точка в пределах 160 символов** → Обрезаем до первой точки + `" Подробности — на {Бренд}."`

3. **Короткий текст (≤160 символов, без точки)** → Добавляем точку (если нет знака препинания) + `" Подробности — на {Бренд}."`

4. **Длинный текст (>160 символов, без точки в начале)** → Мягкое обрезание:
   - Обрезаем до 160 символов
   - Ищем последний пробел в диапазоне [130-160]
   - Обрезаем по этому пробелу
   - Убираем знаки препинания в конце
   - Добавляем `"... Подробности — на {Бренд}."`

### Примеры обработки:

```typescript
// Пример 1: С точкой
"Текст новости. Еще немного текста" 
→ "Текст новости. Подробности — на FashionSite."

// Пример 2: Короткий без точки
"Краткое описание"
→ "Краткое описание. Подробности — на FashionSite."

// Пример 3: Длинный без точки
"Очень длинный текст который превышает лимит в 160 символов и должен быть обрезан по последнему пробелу для красивого отображения в поисковой выдаче и социальных сетях"
→ "Очень длинный текст который превышает лимит в 160 символов и должен быть обрезан по последнему пробелу для красивого отображения в поисковой... Подробности — на FashionSite."
```

## Преимущества реализации

✅ **Единообразие** - все meta-теги следуют единому стилю  
✅ **Динамичность** - реальные цифры материалов на каждой странице  
✅ **SEO-оптимизация** - описания в пределах рекомендуемых 150-160 символов  
✅ **Читаемость** - мягкое обрезание текста по пробелам  
✅ **Брендинг** - название бренда на всех страницах  
✅ **Гибкость** - CMS может переопределить автоматические значения  

---

## Изменение настроек

Для изменения названия бренда или базового URL отредактируйте `/apps/web/src/config/seo.ts`:

```typescript
export const SITE_CONFIG = {
  brandName: 'Ваш бренд',  // Измените здесь
  baseUrl: process.env.PUBLIC_SITE_URL || 'https://yoursite.com',
};
```

Все страницы автоматически обновятся.
