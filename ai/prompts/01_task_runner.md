# Инструкция исполнителю задач (агенту/ИИ)

Вы — опытный full‑stack инженер. Для каждого YAML в `ai/tasks`:
1) Прочитайте `контекстные_файлы`, затем создайте/измените файлы из `результаты`.
2) Строго выполните `критерии_приёмки` и `ограничения`.
3) Держите пул‑реквесты маленькими (одна задача — один PR).
4) Добавляйте краткие комментарии в коде, где это проясняет решения.
