# Стиль кода

- TypeScript в строгом режиме.
- Astro‑компоненты: контентные страницы без клиентских скриптов.
- React — только для островков; гидрация client:visible или client:idle.
- Без внешних CSS‑фреймворков помимо Tailwind; используем CSS‑переменные‑токены.
- Простые и понятные имена; избегайте «магии»; используйте чистые функции.
