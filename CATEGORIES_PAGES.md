# Страницы категорий

## Описание

Добавлена поддержка страниц категорий с автогенерацией URL и SEO метатегов.

## URL структура

Страницы категорий доступны по адресу:
```
/categories/{slug}
```

Где `{slug}` - автоматически генерируемый URL-алиас категории.

## Поля категории в CMS

### 1. Название (name)
- **Тип:** текст
- **Обязательное:** да
- **Уникальное:** да
- **Описание:** Основное название категории

### 2. URL (slug)
- **Тип:** текст
- **Обязательное:** да
- **Уникальное:** да
- **Автогенерация:** да
- **Правило генерации:**
  - Берется `name`
  - Переводится в нижний регистр
  - Удаляются все символы кроме букв, цифр, пробелов и дефисов
  - Пробелы заменяются на дефисы
  - Множественные дефисы объединяются в один

**Пример:**
```
Название: "Мода и Стиль"
Slug: "мода-и-стиль"
URL: /categories/мода-и-стиль
```

### 3. Meta Title (metaTitle)
- **Тип:** текст
- **Автогенерация:** да
- **Шаблон:** `{НАЗВАНИЕ КАТЕГОРИИ} - LADY.NEWS`
- **Редактируемое:** да

**Пример:**
```
Название: "Мода и Стиль"
Meta Title: "Мода и Стиль - LADY.NEWS"
```

### 4. Meta Description (metaDescription)
- **Тип:** textarea
- **Автогенерация:** да
- **Шаблон:** `Все материалы из категории {НАЗВАНИЕ КАТЕГОРИИ}`
- **Редактируемое:** да

**Пример:**
```
Название: "Мода и Стиль"
Meta Description: "Все материалы из категории Мода и Стиль"
```

## Функционал страницы категории

### Отображение статей
- Показываются только статьи выбранной категории
- Статьи фильтруются по `categories.id`
- Статус статьи должен быть `published`
- Сортировка по дате публикации (новые первыми)

### Layout
- Аналогичен главной странице
- Используется компонент `CardB` для карточек статей
- Сетка 1/2/3 колонки (мобильный/планшет/десктоп)

### Breadcrumbs
```
Главная / {Название категории}
```

## Меню навигации

Меню в шапке сайта динамически загружается из CMS:
- Запрос к `/api/categories?limit=10&sort=name`
- Показываются первые 10 категорий по алфавиту
- Каждая категория - ссылка на `/categories/{slug}`

## Миграция базы данных

Файл: `apps/cms/src/migrations/2025-12-10-02-add-categories-fields.ts`

### Добавленные колонки:
1. `slug` (varchar, unique) - URL алиас
2. `meta_title` (varchar) - SEO заголовок
3. `meta_description` (text) - SEO описание

### Обработка существующих категорий:
- Для всех существующих категорий автоматически генерируются:
  - `slug` из поля `name`
  - `meta_title` по шаблону
  - `meta_description` по шаблону

## API

### Получить все категории
```
GET /api/categories?limit=100
```

**Ответ:**
```json
{
  "docs": [
    {
      "id": 1,
      "name": "Мода и Стиль",
      "slug": "мода-и-стиль",
      "metaTitle": "Мода и Стиль - LADY.NEWS",
      "metaDescription": "Все материалы из категории Мода и Стиль"
    }
  ]
}
```

### Получить статьи категории
```
GET /api/articles?where[categories][equals]={categoryId}&where[status][equals]=published&limit=100&sort=-publishedDate
```

## Примеры использования

### Создание новой категории в CMS:

1. Откройте `/admin/collections/categories`
2. Нажмите "Create New"
3. Введите **Название:** "Красота и Здоровье"
4. Автоматически заполнятся:
   - **URL:** `красота-и-здоровье`
   - **Meta Title:** `Красота и Здоровье - LADY.NEWS`
   - **Meta Description:** `Все материалы из категории Красота и Здоровье`
5. При необходимости отредактируйте метатеги
6. Сохраните

### Результат:
- Страница доступна: `/categories/красота-и-здоровье`
- В меню появится ссылка "Красота и Здоровье"
- На странице отобразятся все статьи этой категории

## Рекомендации

### SEO
- Всегда проверяйте автосгенерированные метатеги
- При необходимости редактируйте под конкретную категорию
- Meta Title: до 60 символов
- Meta Description: до 160 символов

### URL slug
- Можно редактировать вручную после создания
- Используйте короткие, понятные URL
- Избегайте спецсимволов
- Используйте дефисы вместо пробелов

### Меню
- Отображаются первые 10 категорий
- Сортировка по алфавиту
- Для контроля порядка можно добавить поле `order` в будущем
