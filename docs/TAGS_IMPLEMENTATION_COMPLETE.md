## 🎨 Особенности реализации

### Автогенерация slug
- Транслитерация кириллицы в латиницу (а→a, я→ya, и т.д.)
- Замена пробелов на дефисы
- Удаление специальных символов
- Примеры:
  - "Мода" → "moda"
  - "Уличная мода" → "ulichnaya-moda"
  - "Красота и стиль" → "krasota-i-stil"

### SEO-оптимизация
- Автоматическая генерация Meta Title: "[Название тега] - LADY.NEWS"
- Автоматическая генерация Meta Description: "Все материалы по тегу [Название]"
- Возможность ручного редактирования в админке
- JSON-LD разметка для поисковых систем

### Единый дизайн
- Облако тегов с rounded-full стилем
- Эффекты hover с плавными переходами
- Адаптивная верстка (grid + flex)
- Соответствие общему стилю сайта

## 🧪 Как проверить

### 1. Админка CMS
```
http://localhost:3000/admin/collections/tags
```
- Создайте новый тег
- Убедитесь, что slug генерируется автоматически
- Проверьте автозаполнение SEO-полей
- Отредактируйте существующий тег

### 2. Страница списка тегов
```
http://localhost:4321/tags/
```
- Должны отображаться все теги
- Каждый тег - кликабельная ссылка
- Красивое облако тегов с hover эффектами

### 3. Страница отдельного тега
```
http://localhost:4321/tags/sport
```
- Отображается название тега
- Показываются все новости с этим тегом
- Хлебные крошки работают
- Если нет новостей - показывается сообщение

### 4. Страница статьи
```
http://localhost:4321/article/...
```
- Внизу статьи отображаются теги
- Клик по тегу ведет на страницу тега
- Теги стильно оформлены

### 5. Навигация
```
http://localhost:4321/
```
- В меню есть ссылка "Теги"
- Клик ведет на /tags/
