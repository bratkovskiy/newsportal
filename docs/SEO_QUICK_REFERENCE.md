# SEO Quick Reference

## Быстрый доступ к SEO-шаблонам

### Конфигурация
- **Файл**: `/apps/web/src/config/seo.ts`
- **Бренд**: `SITE_CONFIG.brandName` (по умолчанию: "FashionSite")

### Примеры результатов

#### Главная страница
```
Title: Последние новости сегодня — 150 материалов | FashionSite
Description: Актуальные события, аналитика и топ-материалы: 150 свежих новостей в России и мире на FashionSite.
```

#### Раздел "Политика" (50 статей)
```
Title: Политика — 50 новостей и статей | FashionSite
Description: 50 свежих материалов по теме «Политика»: события, мнения, хроника на FashionSite.
```

#### Тег "Мода" (25 статей)
```
Title: Мода — 25 материалов, новости и обзор | FashionSite
Description: Подборка материалов по тегу «Мода»: 25 новостей, аналитика, фото и видео на FashionSite.
```

#### Поиск "новый год" (10 результатов)
```
Title: Поиск «новый год» — 10 результатов | FashionSite
Description: Найдено 10 материалов по запросу «новый год»: новости, статьи, мнения на FashionSite.
```

#### Статья "10 трендов модного сезона"
```
Title: 10 трендов модного сезона | FashionSite
Description: В этом сезоне дизайнеры представили множество интересных решений. Подробности — на FashionSite.
```
(Description автоматически обрезается по первой точке или до 150-160 символов)

### Переопределение через CMS

Для страниц категорий и тегов можно задать кастомные SEO-теги через CMS:
- `category.metaTitle` - переопределяет title
- `category.metaDescription` - переопределяет description
- `tag.metaTitle` - переопределяет title
- `tag.metaDescription` - переопределяет description

Если кастомные значения отсутствуют, используются автоматические шаблоны.

### Изменение бренда

Отредактируйте `/apps/web/src/config/seo.ts`:
```typescript
export const SITE_CONFIG = {
  brandName: 'Ваш бренд', // <-- Измените здесь
  baseUrl: process.env.PUBLIC_SITE_URL || 'https://yoursite.com',
};
```

Изменение применится ко всем страницам автоматически.
