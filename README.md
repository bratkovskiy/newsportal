# Newsportal

Монорепозиторий новостного портала.

В составе проекта:

- `apps/cms` — Payload CMS (админка, хранение статей, категорий, тегов, фидов и настроек).
- `apps/web` — фронтенд на Astro (публичный сайт).
- `apps/jobs` — сервис фоновых задач (импорт RSS, cron-автообновление фидов, очереди BullMQ).
- `apps/search` — схема для поискового индекса (Typesense).

## Документация

Подробные технические описания и инструкции находятся в папке `docs/`:

- `docs/DEPLOY.md` — как разворачивать проект (Docker, .env, миграции Payload CMS и т.п.).
- `docs/DOCS.md` — общая документация по RSS-импорту, главной странице, блокам, SEO и мультиязычности.
- Остальные файлы `docs/*.md` — подробные разборы отдельных подсистем (RSS-мэппинг, счётчики просмотров, SEO-шаблоны и др.).

## Быстрый старт (локально через Docker)

1. Установить Docker и Docker Compose.
2. Создать `.env` в корне (см. примеры `.env.example` в `apps/*`).
3. Запустить сервисы:

   ```bash
   docker compose up -d --build
   ```

4. Прогнать миграции CMS (один раз после первого запуска или обновления схемы):

   ```bash
   docker compose exec cms npx payload migrate
   ```

5. Открыть:
   - CMS: http://localhost:3000 (админка Payload);
   - Сайт: http://localhost:4321.

## Автообновление RSS-фидов

Фоновый сервис `apps/jobs` по cron-расписанию импортирует RSS-фиды из CMS.

Кратко:

- Для каждого фида в CMS можно задать поле `cron` (формат Unix-cron из 5 полей).
- Фиды с пустым `cron` **не** импортируются автоматически (только ручной импорт кнопкой в админке).
- Фиды с заполненным `cron` импортируются автоматически через сервис `jobs`.
- Подробности см. раздел "Автообновление RSS-фидов (cron + сервис jobs)" в `docs/DOCS.md`.

## Продакшен

Рекомендуемый способ развёртывания — через `docker-compose` (см. `docs/DEPLOY.md`).

- Все приложения (`cms`, `web`, `jobs`, `postgres`, `redis`) описаны в `docker-compose.yml`.
- Переменные окружения задаются через корневой `.env`.
- Сервис `jobs` стартует вместе с остальными контейнерами и автоматически занимается импортом фидов по cron.
