# Project Brief: modareview.com

**Последнее обновление:** 20 марта 2026

---

## Описание проекта

modareview.com — новостной портал о моде и стиле.

**Основная миссия:** Агрегация и публикация новостей о моде, стиле и трендах из различных источников через RSS-фиды.

---

## Целевая аудитория

**Основная аудитория:**
- Женщины, интересующиеся модой и стилем

## Контентная стратегия

### Источник контента

**Основной источник:** RSS-фиды внешних источников

**Процесс:**
- Настройка RSS-фидов в Payload CMS
- Автоматический импорт по расписанию (cron)
- Ручной импорт через админку при необходимости
- Маппинг полей из RSS в структуру статей

**Темп публикаций:**
- Зависит от активности RSS-источников
- Автоматическая публикация статей с обложками
- Черновики для статей без изображений (требуют ручной обработки)

### Форматы контента

**Основной формат:** Новостные статьи из RSS-фидов

**Структура статьи:**
- Заголовок (из RSS)
- Cover image (из тега `<custom:image>` в RSS)
- Контент (из RSS, поле description или content:encoded)
- Категория (маппинг из RSS или дефолтная)
- Автор (из RSS или назначенный)
- Дата публикации

## Категории на сайте

Категории создаются в Payload CMS и отображаются в навигации сайта (первые 10 по алфавиту).

**Основные категории:**
- Мода
- Стиль
- Красота
- Тренды
- Новости
- Шоппинг
- Celebrity
- Street Style

**Автогенерация полей категорий:**
- `slug` — автоматически из названия (транслитерация, lowercase, дефисы)
- `metaTitle` — шаблон: `{НАЗВАНИЕ КАТЕГОРИИ} - LADY.NEWS`
- `metaDescription` — шаблон: `Все материалы из категории {НАЗВАНИЕ КАТЕГОРИИ}`

**Страницы категорий:**
- URL: `/categories/{slug}`
- Показываются только статьи выбранной категории
- Статус `published`
- Сортировка по дате публикации (новые первыми)

## Технические особенности

**Архитектура:**
- Монорепозиторий (apps/cms, apps/web, apps/jobs)
- Payload CMS для управления контентом
- Astro для frontend
- PostgreSQL для хранения данных
- Redis + BullMQ для фоновых задач

**Автоматизация:**
- Автоматический импорт RSS-фидов по cron-расписанию
- Автогенерация slug для статей и категорий
- Автогенерация SEO-полей для категорий
- Счётчики просмотров статей

**Контроль качества:**
- Статьи без обложки попадают в черновики
- Ручная модерация через Payload CMS админку
- Маппинг полей для каждого RSS-фида
